#########################
User's Guide
#########################

This document is provided for users with instructions to download,
configure, build, and use `PrimoGENI <http://www.primessf.net/PrimoGENI>`_ to develop network models. This
document is generated using `Sphinx <http://sphinx.pocoo.org/>`_ and `Doxygen <http://www.doxygen.org>`_.
Sphinx is a python documentation generator that uses a markup language, called
`reStructuredText <http://docutils.sourceforge.net/rst.html>`_.
Doxygen is a documentation system which is useful for languages such
as C++, C, and Java. 

All source code as well as HTML and PDF versions of this document are
available at `http://www.primessf.net/PrimoGENI <http://www.primessf.net/PrimoGENI/>`_.

If you have trouble downloading, using, or installing PrimoGENI you can email :literal:`help@primessf.net` for assistance.
